Recruitment day

You will need a GitHub account, please setup a public repo to upload your solutions to Problem 1, 2 & 3. Let your facilitator know your repos URL.

When writing your code, please think about SOLID principals and writing good clean code (Readable, Testable, Maintainable, Efficient, Clean)

Session A (Morning to Lunch)

Problem 1

In groups of 2, go to this Batman CodinGame and solve it so all tests pass. Once it passes, upload your solution to your GitHub repo.

Problem 2

Individually, look at Validation, and have a look at the requirement of what you need to add.

Password validation is it needs to have letters, numbers and at least 1 of the following symbols (!@#&*), and it can not have 3 consecuative numbers, example 456.

In your command line, type npm run mocha to run tests.

Session B (Lunch to End)

Problem 3

Look at Optimisation, in Chrome DevTools press Ctrl+Shift+P, type in Performance and select "Drawer Performance Monitor". Then unselect all options and select "Layouts / sec". Get the Layouts a second down to under 60.
